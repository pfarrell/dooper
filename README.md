# Dooper

Exposes some objects that allow simple interaction with S3, WebHcat, and HDFS

## Installation

    gem 'dooper'
    gem install dooper


## Environment vars
  We let this gem pull in credentials and metadata from environment variables

  * AMAZON_SECRET_ACCESS_KEY
  * AMAZON_ACCESS_KEY_ID
  * AWS_REGION
  * WEBHCAT_URL
